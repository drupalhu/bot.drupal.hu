Bot google
==========

This module is an extension of the drupal bot module: http://drupal.org/project/bot. It allows searches be done from IRC. The keyword is s, for example:

<snufkin> s my pony
<bot> My Little Pony => http://www.hasbro.com/mylittlepony/ - more results: http://google.com/search?q=my%20pony
